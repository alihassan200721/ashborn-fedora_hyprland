# Fedora Hyprland Dotfiles

This repository contains my personal **Fedora Hyprland configuration files**, including:

- Hyprland configs (`~/.config/hypr/`)
- Waybar configs (`~/.config/waybar/`)
- Wallpapers

These files are set up for my personal workflow and aesthetic preferences.

## Features

- Fully configured **Hyprland** environment
- Custom **Waybar** modules (workspaces, clock, volume, battery, internet, power)
- Collection of wallpapers

## Usage

You can use these dotfiles as a reference or copy them to your own system:

```bash
git clone https://github.com/<your-username>/fedora-hyprland-dotfiles.git
cd fedora-hyprland-dotfiles
# Copy configs to your ~/.config/
cp -r hypr ~/.config/
cp -r waybar ~/.config/
